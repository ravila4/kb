## Match Location

| Action      | Regex     |
| :---------- | :-------- |
| Match start | `^string` |
| Match end   | `string$` |

## Datatypes

| Action                  | Regex |
| :---------------------- | :---- |
| Match integer           | `\d`  |
| Match multiple integers | `\d+` |
